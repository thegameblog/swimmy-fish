Swimmy Fish
===========

[![Screenshot][]][play]

Just keep swimming. And clicking. Swim over and under obstacles to survive
as long as you can.

[Play it!][play]


Running locally
---------------

1. Make sure [Gesso][] is installed

   ```bash
   $ npm install -g gesso
   ```

2. Install dependencies

   ```bash
   $ npm install
   ```

3. Run the server

   ```bash
   $ gesso serve
   ```

Now visit [http://localhost:63550](http://localhost:63550/).
